# 🍕 Twin Pizza POS - Quick Reference Card

Print this and keep it near the POS terminal!

## 🎯 BASIC OPERATIONS

### Start the App
```
Double-click: START-POS.bat
```

### Take an Order
1. Click **category** on left
2. Click **product** → Wizard opens
3. **Customize** through steps (size, meat, extras, notes, quantity)
4. Click **"Ajouter au panier"** when done
5. Click **"🛒 Panier"** to view cart
6. Click **"Procéder au paiement"** to checkout
7. Enter **customer name** and **phone**
8. Select **payment method** (Espèces/CB)
9. Click **"Valider"** ✅

### ✨ What Happens Automatically
- 🖨️ Ticket prints to thermal printer
- 💬 WhatsApp message sent to customer
- 💵 Cash drawer opens (if cash payment)
- 📱 Order appears in admin dashboard

## 🔧 COMMON TASKS

### Edit Cart
- **Change quantity:** ± buttons
- **Remove item:** ✕ button
- **View total:** Bottom of cart

### Check Status
Look at **top right** for:
- 🟢 = Working
- 🟡 = Connecting
- 🔴 = Error

### Start New Order
After checkout → Automatically returns to catalog

## ⚠️ TROUBLESHOOTING

| Problem | Solution |
|---------|----------|
| **App won't start** | Restart computer, check antivirus |
| **Printer not printing** | Check if printer is on/connected |
| **WhatsApp error** | Check internet, restart app |
| **Phone number error** | Format: `06 XX XX XX XX` or `+33 6 XX XX XX XX` |
| **Product list empty** | Check internet, restart app |

---

## 📞 STATUS INDICATORS (Top Right)

```
🟢 = Connected/Ready
🟡 = Loading/Connecting
🔴 = Offline/Error
⚪ = Unknown
```

**Green** = Everything working  
**Red** = Check internet/devices

## 💾 PAYMENT METHODS

## 🆘 EMERGENCY

**App freezes?**  
```
ALT+F4 to close, then double-click START-POS.bat
```

**Can't reach printer?**  
```
Turn printer off, wait 10 seconds, turn on
```

**Internet down?**  
```
App will pause - restart when internet returns
```
