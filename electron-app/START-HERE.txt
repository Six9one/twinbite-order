HOW TO USE:
──────────────────────────────────────────────────────────────────────────────

STEP 1: INSTALLATION (15 minutes)
   1. Open command prompt in this folder
   2. Run: npm install
   3. Verify no red errors
   4. Done!

STEP 2: CONFIGURATION (5 minutes)
   1. Edit: .env file
   2. Update PRINTER_IP to your printer's IP
   3. Save and close
   4. Done!

STEP 3: LAUNCH
   Option A (Recommended): Double-click START-POS.bat
   Option B (Dev mode):    npm run dev
   Option C (Production):  npm start

STEP 4: TRAIN STAFF
   1. Print: QUICK-REFERENCE.md
   2. Post near POS terminal
   3. Run practice orders together
   4. Done!

TROUBLESHOOTING:
──────────────────────────────────────────────────────────────────────────────

❌ App won't start
   ✅ Fix: npm install again, check Node.js version (18+)

❌ Printer not printing
   ✅ Fix: Check IP in .env, ping printer, restart app

❌ WhatsApp not connecting
   ✅ Fix: Scan QR code on first run, wait 20+ seconds

❌ Products not loading
   ✅ Fix: Check internet, restart app

💡 For detailed help: See SETUP-GUIDE.md Troubleshooting section
